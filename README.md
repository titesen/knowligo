# KnowLigo — RAG WhatsApp Chatbot

KnowLigo es un proyecto educativo que implementa un chatbot RAG
capaz de responder consultas exclusivamente a partir de
documentación corporativa simulada.

### Tecnologías utilizadas
- WhatsApp Cloud API (Meta)
- n8n (local, Docker)
- Ollama (embeddings locales)
- Qdrant (vector database)
- Groq (LLM)

### Objetivo
Demostrar un flujo completo de recuperación de información (RAG),
con control estricto de dominio, uso responsable del modelo
y prevención de abusos.

> Proyecto con fines educativos. No destinado a producción.
