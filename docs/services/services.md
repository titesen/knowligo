# Servicios de KnowLigo

KnowLigo ofrece servicios de soporte técnico y mantenimiento IT orientados
a garantizar la estabilidad y el correcto funcionamiento de los sistemas
informáticos de sus clientes.

---

## Soporte técnico

El servicio de soporte técnico incluye la atención de incidencias relacionadas con:

- sistemas operativos
- software de uso empresarial
- problemas de conectividad de red
- fallas básicas de hardware

El soporte se brinda dentro del horario y alcance definidos por el plan contratado.

---

## Mantenimiento preventivo

El mantenimiento preventivo tiene como objetivo reducir la aparición de incidencias.

Incluye:
- actualizaciones de software
- revisiones periódicas de sistemas
- verificación de copias de seguridad

La frecuencia del mantenimiento depende del plan de servicio.

---

## Gestión de incidencias

Las incidencias se clasifican según su nivel de prioridad:

- Baja: problemas menores sin impacto operativo
- Media: problemas que afectan parcialmente la operación
- Alta: problemas críticos que impiden la operación normal

Los tiempos de respuesta están definidos en el acuerdo de nivel de servicio (SLA).
