# Planes de Servicio de KnowLigo

KnowLigo ofrece tres planes de soporte IT diseñados para cubrir las necesidades de PyMEs de distintos tamaños y niveles de dependencia tecnológica. Todos los planes incluyen acceso al portal de soporte web y aplicación móvil para seguimiento de tickets.

## Plan Básico

- **Precio**: $199.000 ARS/mes + IVA (equivalente aproximado: USD 199/mes)
- **Destinado a**: Empresas de 10 a 30 empleados con infraestructura IT simple
- **Tickets mensuales incluidos**: Hasta 15 tickets por mes
- **Horario de soporte**: Lunes a viernes de 08:00 a 18:00
- **Canales de contacto**: Email, portal web y teléfono
- **Tiempo de respuesta inicial**: Según SLA estándar (prioridad Baja: 24h, Media: 8h, Alta: 4h)
- **Soporte presencial**: No incluido. Disponible con cargo adicional de $25.000 ARS por visita
- **Mantenimiento preventivo**: Trimestral (1 vez cada 3 meses)
- **Monitoreo de servidores**: No incluido
- **Backup gestionado**: No incluido
- **Capacitación**: 1 hora de capacitación inicial al contratar el plan
- **Ejecutivo de cuenta dedicado**: No incluido

### Qué incluye el Plan Básico

- Soporte técnico remoto para estaciones de trabajo y notebooks
- Instalación y configuración de software estándar (Windows, Microsoft Office, antivirus)
- Resolución de problemas de conectividad de red básica (Wi-Fi, Ethernet, impresoras de red)
- Gestión de cuentas de usuario en Active Directory o Microsoft 365
- Asistencia con impresoras, escáneres y periféricos
- Soporte para clientes de correo electrónico (Outlook, Thunderbird)
- Diagnóstico remoto de problemas de rendimiento en equipos

### Qué NO incluye el Plan Básico

- Soporte para servidores (físicos o virtuales)
- Configuración de firewalls o equipos de seguridad perimetral
- Migraciones de sistemas o datos
- Soporte fuera del horario laboral
- Soporte presencial (se cobra aparte)

## Plan Profesional

- **Precio**: $499.000 ARS/mes + IVA (equivalente aproximado: USD 499/mes)
- **Destinado a**: Empresas de 30 a 100 empleados con servidores y red corporativa
- **Tickets mensuales incluidos**: Hasta 40 tickets por mes
- **Horario de soporte**: Lunes a viernes de 08:00 a 20:00, sábados de 09:00 a 13:00
- **Canales de contacto**: Email, portal web, teléfono, WhatsApp y chat en vivo
- **Tiempo de respuesta inicial**: Según SLA estándar con prioridad en la cola
- **Soporte presencial**: 2 visitas mensuales incluidas. Visitas adicionales: $20.000 ARS cada una
- **Mantenimiento preventivo**: Mensual
- **Monitoreo de servidores**: Incluido en horario laboral (lunes a viernes 08:00-20:00)
- **Backup gestionado**: Configuración y verificación de backup semanal incluida
- **Capacitación**: 4 horas trimestrales de capacitación para usuarios
- **Ejecutivo de cuenta dedicado**: Sí, con reunión mensual de seguimiento

### Qué incluye el Plan Profesional

Todo lo del Plan Básico, más:

- Soporte para servidores Windows Server (hasta 3 servidores físicos o virtuales)
- Administración de Active Directory, GPOs y políticas de seguridad
- Gestión de Microsoft 365 (Exchange Online, SharePoint, Teams, OneDrive)
- Configuración y mantenimiento de VPN para trabajo remoto
- Soporte para switches y access points gestionados
- Gestión de antivirus corporativo centralizado
- Informes mensuales de estado de infraestructura
- Soporte para software de gestión ERP/CRM básico (Tango, Contaplus, Colppy)
- Asistencia en configuración de dispositivos móviles corporativos

### Qué NO incluye el Plan Profesional

- Soporte para más de 3 servidores (se cotiza como adicional)
- Implementación de soluciones cloud desde cero (migraciones se cotizan aparte)
- Desarrollo de software a medida
- Soporte 24/7 (disponible solo en Plan Empresarial)

## Plan Empresarial

- **Precio**: $999.000 ARS/mes + IVA (equivalente aproximado: USD 999/mes)
- **Destinado a**: Empresas de 100 a 200 empleados con infraestructura compleja y alta dependencia tecnológica
- **Tickets mensuales incluidos**: Ilimitados
- **Horario de soporte**: 24 horas, 7 días a la semana, los 365 días del año
- **Canales de contacto**: Todos los canales disponibles + línea directa de emergencia
- **Tiempo de respuesta inicial**: SLA premium (prioridad Crítica: 30 min, Alta: 2h, Media: 4h, Baja: 8h)
- **Soporte presencial**: Ilimitado dentro de CABA y GBA
- **Mantenimiento preventivo**: Semanal
- **Monitoreo de servidores**: 24/7 con alertas automáticas y respuesta proactiva
- **Backup gestionado**: Backup diario con retención de 30 días y pruebas de restauración mensuales
- **Capacitación**: 8 horas mensuales de capacitación para usuarios y equipo técnico interno
- **Ejecutivo de cuenta dedicado**: Sí, con reunión quincenal de seguimiento y reportes ejecutivos

### Qué incluye el Plan Empresarial

Todo lo del Plan Profesional, más:

- Soporte ilimitado para servidores (físicos, virtuales y cloud)
- Administración de infraestructura en Azure, AWS o Google Cloud
- Gestión avanzada de firewalls Fortinet, Cisco y pfSense
- Implementación y gestión de soluciones de virtualización VMware y Hyper-V
- Plan de recuperación ante desastres (DRP) documentado y probado semestralmente
- Auditorías de seguridad trimestrales con informe ejecutivo
- Soporte para bases de datos SQL Server, PostgreSQL y MySQL
- Gestión de certificados SSL y seguridad web
- Soporte para telefonía IP y sistemas de videoconferencia
- Prioridad máxima en la cola de atención

## Condiciones generales de contratación

- **Período mínimo de contratación**: 6 meses para todos los planes
- **Forma de pago**: Transferencia bancaria o débito automático. Pago mensual por adelantado.
- **Facturación**: Se emite factura electrónica dentro de los primeros 5 días hábiles del mes.
- **Cancelación**: Con aviso previo de 30 días antes del fin del período actual. No se realizan reembolsos por meses parciales.
- **Cambio de plan**: Se puede realizar al finalizar cualquier mes calendario con 15 días de aviso previo. El upgrade es inmediato; el downgrade se aplica al mes siguiente.
- **Excedentes de tickets**: En planes Básico y Profesional, los tickets que excedan el cupo mensual se cobran a $8.000 ARS cada uno.
- **Garantía de satisfacción**: Si el cliente no está satisfecho durante los primeros 30 días, puede cancelar sin penalidad.
