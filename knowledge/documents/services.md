# Servicios de KnowLigo

## Soporte técnico remoto

El soporte técnico remoto es el servicio principal de KnowLigo. Mediante herramientas de acceso remoto seguro (AnyDesk y Microsoft Remote Desktop), nuestros técnicos se conectan al equipo del usuario para resolver incidencias sin necesidad de desplazamiento.

### Alcance del soporte remoto

- Diagnóstico y resolución de problemas de software (sistema operativo, aplicaciones de ofimática, navegadores, clientes de correo)
- Instalación, actualización y desinstalación de software autorizado
- Configuración de impresoras, escáneres y periféricos de red
- Resolución de problemas de conectividad (red local, Wi-Fi, VPN)
- Gestión de cuentas de usuario (creación, modificación, reseteo de contraseñas, permisos)
- Optimización de rendimiento de equipos (limpieza de archivos temporales, gestión de inicio, actualizaciones del sistema)
- Asistencia con herramientas de colaboración (Microsoft Teams, Google Workspace, Zoom)
- Configuración de firma de correo electrónico corporativa
- Soporte para dispositivos móviles corporativos (configuración de email, VPN, políticas de seguridad)

### Sistemas operativos soportados

- Windows 10 y Windows 11 (ediciones Pro y Enterprise)
- Windows Server 2016, 2019 y 2022
- macOS (soporte limitado a aplicaciones de productividad, no a nivel de sistema)
- Linux (Ubuntu Server y CentOS para servidores, soporte de Nivel 2 y 3)

### Software soportado

- Microsoft Office 365 y versiones perpetuas (2019, 2021)
- Navegadores: Google Chrome, Microsoft Edge, Mozilla Firefox
- Antivirus: ESET, Kaspersky, Windows Defender (gestión centralizada)
- Herramientas de gestión: Tango Gestión, Colppy, Xubio
- Software de diseño y multimedia: Solo instalación y configuración básica (no soporte funcional)

## Soporte técnico presencial

El soporte presencial se brinda en las oficinas del cliente dentro de la zona de cobertura (CABA y GBA). Incluye:

- Reemplazo y diagnóstico de hardware (discos, memorias, fuentes de poder, placas de red)
- Cableado de red estructurado y certificación de puntos de red
- Instalación física de servidores, switches, routers y access points
- Reubicación de equipos y puestos de trabajo
- Auditorías de infraestructura física (relevamiento de equipos, etiquetado, inventario)

El soporte presencial está incluido según el plan contratado. Para el Plan Básico, las visitas tienen un costo adicional de $25.000 ARS por visita.

## Administración de servidores

KnowLigo administra servidores físicos y virtuales para clientes de los planes Profesional y Empresarial.

### Tareas de administración incluidas

- Instalación y configuración inicial de Windows Server, Linux (Ubuntu, CentOS)
- Administración de Active Directory: creación de unidades organizativas, usuarios, grupos, políticas de grupo (GPO)
- Gestión de servicios de directorio: DNS, DHCP, archivos compartidos (SMB/CIFS)
- Administración de Microsoft 365: Exchange Online, SharePoint, OneDrive, Teams
- Monitoreo de salud del servidor: CPU, memoria, disco, servicios críticos
- Aplicación de parches y actualizaciones de seguridad mensuales
- Gestión de certificados SSL/TLS
- Administración de bases de datos SQL Server, PostgreSQL y MySQL (tareas básicas de mantenimiento: backups, optimización de índices, monitoreo de espacio)

### Virtualización

- Implementación y gestión de entornos VMware ESXi y Microsoft Hyper-V
- Creación y administración de máquinas virtuales
- Migración P2V (Physical to Virtual) y V2V (Virtual to Virtual)
- Snapshots y planes de contingencia para actualizaciones críticas

## Gestión de redes y seguridad

### Redes

- Diseño e implementación de redes LAN y WLAN corporativas
- Configuración de switches gestionados (Cisco, HP/Aruba, Ubiquiti)
- Implementación de VLANs para segmentación de red
- Configuración de Access Points empresariales con portal cautivo
- Implementación y gestión de VPN site-to-site y client-to-site
- Diagnóstico y resolución de problemas de red con herramientas especializadas

### Seguridad perimetral

- Configuración y gestión de firewalls (Fortinet FortiGate, Cisco ASA, pfSense)
- Implementación de reglas de filtrado, NAT y port forwarding
- Configuración de IDS/IPS (Intrusion Detection/Prevention Systems)
- Gestión de antivirus corporativo centralizado
- Implementación de políticas de seguridad de endpoints
- Auditorías de seguridad trimestrales (disponible en Plan Empresarial)

## Mantenimiento preventivo

El mantenimiento preventivo es un servicio proactivo diseñado para prevenir fallas y garantizar el rendimiento óptimo de la infraestructura.

### Actividades incluidas en el mantenimiento preventivo

- Verificación del estado de hardware: discos (SMART), fuentes, ventiladores, temperaturas
- Actualización de sistemas operativos y parches de seguridad
- Actualización de firmware de equipos de red (switches, routers, firewalls, access points)
- Revisión y optimización de Active Directory y políticas de grupo
- Limpieza de logs y archivos temporales en servidores
- Verificación de integridad de backups y prueba de restauración selectiva
- Revisión de licencias de software y fechas de vencimiento
- Control de espacio en disco en servidores y estaciones críticas
- Revisión de antivirus: estado de actualizaciones, análisis programados, amenazas detectadas
- Generación de informe de mantenimiento con hallazgos y recomendaciones

### Frecuencia del mantenimiento preventivo

| Plan | Frecuencia | Alcance |
|---|---|---|
| Básico | Trimestral | Estaciones de trabajo únicamente |
| Profesional | Mensual | Estaciones de trabajo + servidores + red |
| Empresarial | Semanal | Toda la infraestructura + auditoría de seguridad |

## Backup y recuperación ante desastres

### Servicio de backup gestionado

- Configuración de políticas de backup automatizado (Veeam Backup & Replication)
- Backup de servidores completos (imagen de sistema)
- Backup de archivos y carpetas compartidas
- Backup de bases de datos (SQL Server, PostgreSQL)
- Backup de buzones de correo en Microsoft 365
- Retención configurable: 7, 15 o 30 días según plan
- Almacenamiento en repositorio local y/o nube (Azure Blob Storage)

### Plan de recuperación ante desastres (DRP)

Disponible exclusivamente para clientes del Plan Empresarial:

- Documentación completa del plan de contingencia
- Definición de RTO (Recovery Time Objective) y RPO (Recovery Point Objective)
- Procedimientos paso a paso para restauración de servicios críticos
- Pruebas de recuperación semestrales con informe de resultados
- Failover a entorno secundario para servicios críticos

## Gestión de incidencias y tickets

### Proceso de atención de incidencias

1. **Apertura del ticket**: El usuario reporta la incidencia a través de cualquier canal habilitado (portal web, email, teléfono, WhatsApp).
2. **Registro y clasificación**: Se asigna un número de ticket, se clasifica la prioridad (Baja, Media, Alta, Crítica) y se asigna al técnico correspondiente.
3. **Diagnóstico inicial**: El técnico contacta al usuario dentro del tiempo de respuesta establecido en el SLA.
4. **Resolución**: Se trabaja en la solución y se mantiene informado al usuario del avance.
5. **Cierre**: Se confirma la resolución con el usuario y se cierra el ticket. Se envía una encuesta de satisfacción.
6. **Seguimiento**: Si el problema reaparece dentro de las 48 horas siguientes, se reabre el mismo ticket sin consumir cupo.

### Clasificación de prioridades

| Prioridad | Descripción | Ejemplo |
|---|---|---|
| Baja | Incidencia menor que no afecta la operación | Solicitud de instalación de software, consulta general |
| Media | Incidencia que afecta a un usuario pero tiene solución temporal | Problema con impresora, lentitud en un equipo |
| Alta | Incidencia que afecta a múltiples usuarios o un servicio importante | Servidor de archivos caído, email corporativo inaccesible |
| Crítica | Incidencia que detiene completamente la operación del negocio | Todos los sistemas caídos, ataque de ransomware, pérdida de datos |

La prioridad Crítica está disponible exclusivamente para clientes del Plan Empresarial.
